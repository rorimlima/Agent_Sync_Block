Política de Privacidade

Última atualização: 15 de maio de 2026

1. Informações Coletadas

O Agent Sync Block coleta apenas as informações necessárias para o funcionamento do sistema:

  • Dados de autenticação: Nome, e-mail e credenciais de acesso para login seguro.
  • Dados operacionais: Registros de veículos, clientes e transações inseridos pelos usuários no exercício de suas funções.
  • Logs de atividade: Registros de ações realizadas no sistema para fins de auditoria e segurança.

2. Uso das Informações

As informações coletadas são utilizadas exclusivamente para:

  • Gerenciar bloqueios e desbloqueios de veículos.
  • Controlar inadimplência e situação financeira de clientes.
  • Sincronizar dados entre dispositivos de forma segura.
  • Gerar relatórios e auditorias internas.

3. Armazenamento e Segurança

Os dados são armazenados de forma segura utilizando:

  • Supabase: Banco de dados na nuvem com criptografia em trânsito (TLS) e em repouso, com isolamento por tenant via Row Level Security (RLS).
  • IndexedDB local: Cache local criptografado no dispositivo para funcionamento offline, com limpeza automática de dados expirados.
  • Autenticação JWT: Tokens de acesso com expiração controlada para sessões seguras.

4. Compartilhamento de Dados

O Agent Sync Block não vende, aluga ou compartilha dados pessoais com terceiros. Os dados são acessíveis apenas pelos usuários autorizados dentro da organização contratante.

5. Retenção de Dados

  • Dados ativos são mantidos durante a vigência do serviço.
  • Registros finalizados são removidos do cache local após 15 dias.
  • Registros excluídos (soft-delete) são removidos do cache local após 7 dias.
  • Dados no servidor são mantidos conforme políticas da organização contratante.

6. Direitos do Usuário

O usuário tem direito a:

  • Solicitar acesso aos seus dados pessoais.
  • Solicitar correção de dados incorretos.
  • Solicitar exclusão de seus dados, sujeito a requisitos legais de retenção.
  • Revogar consentimento a qualquer momento.

7. Cookies e Tecnologias de Rastreamento

O Agent Sync Block não utiliza cookies de rastreamento ou analytics de terceiros. Apenas cookies funcionais são utilizados para manter sessões de autenticação ativas.

8. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Alterações significativas serão comunicadas através do aplicativo.

9. Contato

Para dúvidas sobre esta política de privacidade ou sobre o tratamento de dados, entre em contato com o administrador do sistema através do painel de gerenciamento.
